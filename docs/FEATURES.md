# Fitur Chattatan

Dokumen ini menjelaskan fitur yang ada di source code.

## Akun
- Login email/nomor HP + password.
- Register nama, nomor HP, email, tanggal lahir, status, password.
- Edit profil lengkap.
- Upload foto profil.
- Ganti password dari halaman profil.

## Kontak
- Kontak milik masing-masing akun.
- Tambah kontak tanpa email, cukup nama, nomor HP, status.
- Edit dan hapus kontak.
- Info kontak.
- Nama kontak bisa berbeda di tiap akun.

## Chat
- Private chat memakai user_id, bukan nama kontak.
- Reply message.
- Delivered/read receipt.
- Pin chat.
- Archive chat.
- Attachment gambar dan file.
- Encrypted body sederhana di kolom encrypted_body.

## Grup
- Buat grup.
- Lihat info grup.
- Tambah anggota grup pakai nomor HP.
